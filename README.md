# After Framework — afterframework.com

Public site for **After Framework**: the shared Super App OS behind AfterArtificial products.

Built by [Overstein Labs](https://www.overstein.com). Products live on [AfterArtificial](https://www.afterartificial.com).

## Local

```bash
npm install
npm run dev
```

## Deploy

Vercel project: `afterframework` · domain: `www.afterframework.com`

```bash
vercel --prod
vercel domains add afterframework.com
vercel domains add www.afterframework.com
```
